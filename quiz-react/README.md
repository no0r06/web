# React Quiz App (TypeScript)

## 📌 Project Overview

This project is a simple React Quiz Application built using TypeScript.
The purpose of this project is to practice React setup, Git workflow, and project recovery using version control.

---

## ⚙️ Tech Stack

* React
* TypeScript
* Vite
* Git & GitHub

---

## 🚀 Setup Process

### 1. Create Project

```bash
npm create vite@latest react-quiz
cd react-quiz
npm install
npm run dev
```

---

### 2. Initialize Git & Push to GitHub

```bash
git init
git add .
git commit -m "Initial commit"

git remote add origin https://github.com/YOUR_USERNAME/react-quiz.git
git branch -M main
git push -u origin main
```

---

### 3. Delete Local Project

```bash
cd ..
rm -rf react-quiz
```

---

### 4. Clone Project Again

```bash
git clone https://github.com/YOUR_USERNAME/react-quiz.git
cd react-quiz
npm install
npm run dev
```
